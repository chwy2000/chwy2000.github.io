Instrucciones para GitHub Pages

Para resolver el error de Jekyll, crea un archivo vacío llamado .nojekyll en la raíz de tu repositorio.

Puedes hacerlo con estos comandos:

echo "" > .nojekyll
git add .nojekyll
git commit -m "Add .nojekyll to disable Jekyll processing"
git push

O mediante la interfaz web de GitHub:

  1. Ve a tu repositorio en GitHub
  2. Haz clic en "Add file" → "Create new file"
  3. Nombra el archivo .nojekyll
  4. Deja el contenido vacío y haz commit

Esta aplicación utiliza TensorFlow.js para procesar archivos CSV directamente en el navegador.

Para usar en GitHub Pages, guarda este código como index.html en tu repositorio.
